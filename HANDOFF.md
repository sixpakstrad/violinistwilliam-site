# William Samorey Website Handoff

This is a Next.js App Router website for professional violinist William Samorey and Winspiration Studio LLC.

## Current Structure

- `app/page.tsx` - Main landing page
- `app/about/page.tsx` - About William
- `app/music/page.tsx` - Song library page
- `app/groups/page.tsx` - Ensemble/group options
- `app/rates/page.tsx` - Rates information
- `app/contact/page.tsx` - Contact and inquiry form
- `components/` - Reusable visual sections and navigation
- `data/songs.ts` - Song library data

## Tech Stack

- Next.js App Router
- React
- Tailwind CSS
- Framer Motion
- TypeScript

## Run Locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Next.js, usually:

```text
http://localhost:3000
```

## Notes

- The design currently uses cinematic placeholder media areas for photos and videos.
- Social links are placeholders except for `https://www.winspirationstudio.com`.
- The site is structured as a multi-page performance-marketing website.
