Winspiration Studio Website Preview

Use this same address every time:
http://127.0.0.1:3030

To start the preview:
1. Double-click start-preview.command
2. Keep the Terminal window open
3. Open or refresh http://127.0.0.1:3030

What changed:
The preview now runs the live Next.js website instead of a frozen static copy.
That means new edits should appear after refreshing the browser.

If the page will not load:
1. Close any old Terminal window running the preview
2. Double-click start-preview.command again

If Safari still shows an old version:
1. Quit the preview Terminal window
2. Double-click start-preview.command again
3. Refresh http://127.0.0.1:3030

The preview file now always uses port 3030.
